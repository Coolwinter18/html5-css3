Marcadores de Fuentes

B: se utiliza para poner negritas al texto

BIG: se utiliza para convertir a una fuente más grande

I: se utiliza para convertir el texto a cursiva

S: se utiliza para poner una raya al texto

SMALL: se utiliza para hacer le texto más pequeño

STRIKE: Es el mismo resultado que s, pone una raya en el texto

TT: se utiliza para dar formato de teletipo como se utiliza en los monitores e impresoras antiguas

U: se utiliza para subrayar una palabra
